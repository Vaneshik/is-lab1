# Frontend - Person Management System

React приложение для управления данными.

## Запуск

```bash
npm install
npm start
```

Приложение откроется на `http://localhost:3000`

## Сборка

```bash
npm run build
```

Результат в директории `build/`

## Структура

- `src/pages/` - страницы приложения
- `src/services/` - API клиенты
- `src/App.js` - главный компонент с роутингом
